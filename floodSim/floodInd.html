About:

This is a non-recursive breadth and depth first implementation of floodfill and shortest paths.

Instructions:

Click on start to start. First go to the first menu button to draw walls. If you only want bfs or dfs, don't pick a ending point. Once ready, go to action and choose your prefered action, note, if no starting/ending points are chosen, default ones are chosen for you. Change the board size or filling speed in the settings menu.
