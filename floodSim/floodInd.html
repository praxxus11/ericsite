About:

This is a non-recursive breadth first implementation of floodfill. I first tried a depth-first approach but the recursive function calls in javascript were hard to manage so I didn't want to deal with it.

Instructions:

First click on the red button to start the game. Then click on draw barriers to draw some walls, try to include holes and enclosed areas. Once all walls are drawn, pick ONLY ONE starting point. If you choose multiple, then the program will just choose the last one drawn. Once ready, click on FILL to start! Make sure not to click on restart during the filling process.

You can continue to draw initialization points after the filling is complete.
